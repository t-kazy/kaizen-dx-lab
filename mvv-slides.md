各スライドのテキスト・スピーカーノート（話す内容のメモ）をまとめています。

## スライド 3 — なぜ今、MVVを作ったのか

**テキスト**
```

1. 組織の軸づくり
   → 事業所が違っても、同じ価値観で支援する

2. 採用・ブランディングの土台
   → カイトを知ってもらい、ファンになってもらう

   → 判断に悩んだとき、立ち返れる言葉を持つ

## スライド 4 — MVVとは

**テキスト**
```
MVVとは

MISSION（使命）  ← なぜカイトは存在するのか
    ↓
VISION（目指す未来）← その使命を果たした先に見える景色
    ↓
VALUES（行動指針） ← 日々、どう動くか
```

**スピーカーノート**
3つの関係をまず押さえましょう。MISSIONが一番上にある「存在意義」。VISIONがその先に見える「未来像」。VALUESが日々の「行動指針」。上から下に向かって、抽象→具体になっています。

## スライド 5 — MISSION

**テキスト**
```
MISSION（使命）

子どもの未来を"ソウゾウ"する
```

**スピーカーノート**
カイトの使命を一言にしました。「ソウゾウ」にはあえてカタカナを使っています。二つの意味を込めています。

---

## スライド 6 — MISSIONの解説

**テキスト**
```
"ソウゾウ"に込めた二つの意味

想像する — その子の未来を思い描く
創造する — その子と一緒につくっていく

想像だけでは絵空事。
創造だけでは押しつけ。
両方があってはじめて、
子どもと一緒に未来を形にしていける。
```

**スピーカーノート**
「この子は将来こうなれるかもしれない」と想像する。でもそれだけでは意味がない。実際に一緒に積み上げていくことで初めて形になる。皆さんが日々やっている療育は、まさにこの「ソウゾウ」そのものです。

---

## スライド 7 — VISION

**テキスト**
```
VISION（目指す未来）

"できる"が翼になる。
出会えた一人ひとりが、
自分のチカラで飛べるように。
```

**スピーカーノート**
MISSIONを果たした先に見える景色がこれです。日々の療育で積み上げる小さな「できる」が、やがてその子の「翼」になる。そして、自分のチカラで飛んでいける。カイトの支援のゴールは「自立」です。

---

## スライド 8 — VISIONの解説

**テキスト**
```
"できる"が翼になる

できる → 翼 → チカラ → 飛べる

「すべての子ども」ではなく
「出会えた一人ひとり」。
カイトに来てくれた子への約束。

カイトは飛ばす側ではない。
飛べるように支える側。
主役は、いつも子ども。
```

**スピーカーノート**
「すべての子ども」だと風呂敷が広すぎる。カイトに来てくれた子に対して、誠実に向き合うという約束です。そしてカイトの名前——凧——と「翼」「飛ぶ」は自然につながっている。凧が自分の風をつかんで飛ぶように、子どもが自分のチカラで飛んでいける未来を目指します。

---

## スライド 9 — VALUES導入

**テキスト**
```
VALUES（行動指針）

## スライド 11 — 備（ソナエル）

**テキスト**
```
1. 備 ーソナエルー

すべては「備え」から始まる。

仕事のやり方として
先を読み、段取りをつくる。
「あのとき準備しておけば」をなくすために、
今日できることを今日やる。

子どもたちへの接遇として
安心して過ごせる場を整える。
子ども自身が「自分で備える力」を
身につけられるよう、一緒に練習していく。
```

**スピーカーノート**
カイトが全員正社員にこだわる理由と直結しています。良い支援のためには、事前の準備がとにかく重要。子どもたちが来るまでにしっかり準備して、綿密な計画のうえで療育に臨む。これがカイトの強さの土台です。

---

## スライド 12 — 灯（トモス）

**テキスト**
```
2. 灯 ートモスー

自分の中に火を灯し、誰かの光になる。

仕事のやり方として
「なぜこの仕事を選んだのか」を忘れない。
忙しさの中でも、自分の志や情熱を灯し続ける。
その火は、チームにも静かに伝わっていく。

子どもたちへの接遇として
子どもの中にある小さな火種
——興味、好奇心、「好き」の気持ち——
を見逃さない。消さない。
自分で燃やせるようになるまで見守る。
```

**スピーカーノート**
皆さんがこの仕事を選んだ理由がある。忙しい日もあるけれど、その原点の火を消さないでほしい。そして、子どもの中にある小さな火種を見つけて、そっと風を送る。それが「灯す」ということです。

---

## スライド 13 — 繋（ツナグ）

**テキスト**
```
3. 繋 ーツナグー

一人で抱えず、繋がる。

仕事のやり方として
情報を共有し、チームで支え合う。
「自分だけが知っている」状態をつくらない。

子どもたちへの接遇として
子どもが人を信じ、
人と繋がれる経験を重ねていけるように。
地域や社会との橋渡し役になる。
```

**スピーカーノート**
カイトの名前の由来を思い出してください。凧の糸を、保護者と事業所でしっかり握る。子ども・親・カイトがつながり続ける。一人で抱え込まず、チームで支え合う文化をこれからも大事にしていきたい。

---

## スライド 14 — 起（オコス）

**テキスト**
```
4. 起 ーオコスー

自ら動き、変化を起こす。

仕事のやり方として
指示を待つのではなく、
自分で課題を見つけ、声を上げる。
「気づいた人が動く」文化をつくる。

子どもたちへの接遇として
子どもの「やってみたい」を引き出し、背中を押す。
失敗しても大丈夫だと伝え、
自分で一歩を踏み出す経験を一緒につくっていく。
```

**スピーカーノート**
カイトはボトムアップの組織です。現場への権限移譲をしている。だからこそ、皆さん一人ひとりが「気づいた人が動く」を体現してほしい。子どもに対しても同じで、「やってみたい」を引き出して、一歩踏み出す経験をつくる。

---

## スライド 15 — 拓（ヒラク）

**テキスト**
```
5. 拓 ーヒラクー

まだ誰も歩いていない道を拓く。

仕事のやり方として
「今までこうだったから」にとらわれない。
新しいやり方、新しい視点を試す勇気を持つ。

子どもたちへの接遇として
この子の可能性はこんなものじゃない
——そう信じ続ける。
選択肢を広げ、可能性の扉を一緒に開ける。
```

**スピーカーノート**
IT・DX化の推進もこの精神から来ています。そして将来、製造業と福祉をかけ合わせた就労支援の展開も見据えている。子どもたちが大人になったとき、帰ってこられる場所を作りたい。まだ誰も歩いていない道を、一緒に拓いていきましょう。

## スライド 17 — 最後に

**テキスト**
```
MVVは壁に貼って終わりではなく、
迷ったとき、判断に悩んだときに
立ち返れる言葉にしたい。

皆さんと一緒にこの言葉を育てていきます。
```
