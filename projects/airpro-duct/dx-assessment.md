## 3. 現状 → 理想 → ギャップ分析

### フレームワーク

```
【現状】          【ギャップ】           【理想】
紙・口頭・属人  →  仕組み化が必要  →  デジタルで情報が残り、
                                      誰でもアクセスできる状態
```

### 想定される3大ギャップ

#### ギャップ①：情報がストックされない
- **現状**: 電話・口頭・LINEで流れていく情報
- **理想**: 必要な情報が検索・参照できる状態
- **埋め方**: Google Workspace導入（Chat / Drive / Sheets）

#### ギャップ②：繰り返し作業が手動
- **現状**: 見積・日報・安全書類を毎回ゼロから作成
- **理想**: テンプレート＋自動化で作業時間を削減
- **埋め方**: Google Sheetsテンプレート → 段階的にAI活用

#### ギャップ③：言語・知識の壁
- **現状**: ベテランの暗黙知、技能実習生との言語障壁
- **理想**: 誰でも・どの言語でも必要な情報にアクセスできる
- **埋め方**: AIチャットボット（Dify）で社内QA＋多言語対応

## 4. ソリューション提案（あなたのスキルセットで実現可能なもの）

### 🟢 Phase 1：すぐ着手できる（1〜2ヶ月）

#### 1-A. Google Workspace 基盤整備
- **何をする**: 会社用Googleアカウント → Gmail / Drive / Chat / Sheets導入
- **効果**: 情報がクラウドに残る、どこからでもアクセス可能
- **あなたのスキル**: Google Workspace一通り使える ✅
- **コスト**: Business Starter ¥680/ユーザー/月

#### 1-B. 社内QAチャットボット（Dify）
- **何をする**: よくある質問（作業手順・安全ルール・会社ルール）をAIが回答
- **効果**: 「〇〇さんに聞かないとわからない」が減る
- **あなたのスキル**: Difyチャットフロー作成済み ✅
- **拡張**: 多言語対応で技能実習生もすぐ使える
- **ポイント**: 先方が興味を示した機能→最初の成功体験に最適

### 🟡 Phase 2：基盤ができたら（3〜6ヶ月）

#### 2-A. デジタル日報・作業報告
- **何をする**: Google Formsで現場から報告 → Sheetsに自動集計
- **効果**: 紙の転記作業がゼロに、データが蓄積される
- **発展形**: 写真添付、位置情報付き

#### 2-B. 見積テンプレート自動化
- **何をする**: Google Sheetsでダクト工事の見積テンプレート
- **効果**: 過去の単価参照、計算自動化で見積時間短縮
- **発展形**: AIで見積書の下書き生成

#### 2-C. 技能実習生向け多言語マニュアル
- **何をする**: 作業手順書をAI翻訳＋チャットボットで提供
- **効果**: ベトナム語・中国語等で作業手順を確認可能
- **あなたのスキル**: Dify + ChatGPT/Gemini活用 ✅

### 🔵 Phase 3：データ活用（6ヶ月〜）

#### 3-A. ナレッジベース構築
- **何をする**: ベテランの知識をドキュメント化 → AI検索
- **効果**: 退職リスク軽減、新人教育の効率化
- **あなたのスキル**: Claude Code + GitHub連携でナレッジ管理 ✅

#### 3-B. 業務ダッシュボード
- **何をする**: 工事件数・売上・稼働状況を可視化
- **効果**: 経営判断に使えるデータが見える化
