## 3. 原稿改善サイクル（PDCA）

### 3-1. 基本方針

原稿は「出して終わり」ではなく、**2週間ごとに数値を見て改善**する。

### 3-2. 改善サイクル

```
┌─────────────────────────────────────────┐
│  2週間サイクル                            │
│                                          │
│  Week 1: 掲載 → 数値計測                  │
│  Week 2: 数値分析 → 原稿修正 → 再掲載     │
│                                          │
│  見る数値：                               │
│  ・表示回数（インプレッション）              │
│  ・クリック率（CTR）                       │
│  ・応募数                                 │
│  ・応募単価（費用 ÷ 応募数）               │
└─────────────────────────────────────────┘
```

### 3-3. ABテストの進め方

**テストする要素（優先順）：**

| 優先度 | テスト対象 | 例 |
|---|---|---|
| 1 | 職種名（タイトル） | 「保育士」vs「保育士/放課後デイ」vs「保育士/残業ゼロ」 |
| 2 | キャッチコピー | 「人間関係で選ぶなら」vs「離職率0%の職場」 |
| 3 | 給与の見せ方 | 「月給19.6万円〜」vs「年収274万円〜」vs「時給換算1,167円」 |
| 4 | 写真・画像 | スタッフの笑顔 vs 施設の外観 vs 子どもとの活動風景 |

**ルール：**
- 一度に変える要素は1つだけ
- 最低1週間は同じ原稿で計測する
- 数値が改善した方を残し、次の要素をテスト

### 3-4. 原稿修正のタイミング

| 時期 | やること |
|---|---|
| **2月下旬（今すぐ）** | 現行原稿をIndeedカテゴリ版に差し替え。Indeed有料化を開始 |
| **3月第1週** | 2週間の数値を確認。職種名のABテスト開始 |
| **3月第3週** | ABテスト結果を反映。応募がなければキャッチコピー変更 |
| **4月以降（採用できなかった場合）** | 給与・条件の見直しを検討。媒体追加も視野に |

## 4. SNS・採用広報戦略

### 4-1. Instagram運用

**目的：** 「この職場で働きたい」と思わせる空気感を伝える

**投稿頻度：** 週2〜3回

**コンテンツカレンダー（週間）：**

| 曜日 | 投稿テーマ | 例 |
|---|---|---|
| 月 | スタッフの日常 | 「朝ミーティングの様子」「今日のランチ」 |
| 水 | 療育・活動の紹介 | 「カイトひろばで運動療育」「すららで学習支援」 |
| 金 | 職場の雰囲気・想い | 「入社して感じたこと（スタッフの声）」「なぜこの仕事を選んだか」 |

**投稿のポイント：**
- 顔出しNGなら後ろ姿・手元でOK
- 「映え」より「リアル感」を重視
- ストーリーズで日常を気軽に発信（残業なしで帰る様子、駐車場の風景など）
- プロフィール欄に「採用情報はハイライトへ」と導線を作る

**採用ハイライト（固定）：**
1. 募集要項（Indeed原稿の要約）
2. 1日の流れ
3. スタッフの声
4. 見学案内

### 4-2. 採用広報で伝えるべき「3つの柱」

求人原稿・SNS・面接すべてで一貫して伝えるメッセージ：

1. **人間関係の良さ**（離職率0%・何でも言い合える）
2. **本当の残業ゼロ**（持ち帰りなし・19時退勤）
3. **未経験でも安心の体制**（OJT・チーム制）

## 5. 面接・選考プロセス

### 5-1. 選考フロー（現行の改善案）

```
応募 → 3日以内に連絡 → 見学・面談（1回目）→ 面接（2回目）→ 内定 → 入社
         ↑ここが最重要               ↑省略可能
```

**即レスの徹底：**
- 応募から**24時間以内**に初回連絡（電話 or メッセージ）
- 72時間以上空くと応募者は他社に流れる
- 土日の応募は月曜午前中に対応

**見学と面接を分ける理由：**
- 1回目は「見学＋カジュアル面談」→ 応募者の心理的ハードルを下げる
- 2回目で「面接」→ お互いのマッチングを確認
- 急ぎの場合は1回にまとめてもOK

### 5-2. 面接で確認すべきポイント

| 確認項目 | 質問例 |
|---|---|
| チームワーク志向 | 「前の職場でチームで取り組んだ経験を教えてください」 |
| 退職理由の本音 | 「前の職場で一番しんどかったことは？」 |
| 送迎への抵抗感 | 「普段の運転頻度は？送迎について不安はありますか？」 |
| 長期定着の意思 | 「3年後、どんな働き方をしていたいですか？」 |

### 5-3. 面接後のフォロー

- 面接後**3日以内**に合否連絡
- 内定者には入社前に「1日体験」を提案（任意）
- 入社日までの期間が長い場合は月1回の連絡で関係維持

## 6. 定着施策

**せっかく採用しても辞められたら意味がない。**

### 入社後90日プログラム

| 時期 | 施策 |
|---|---|
| 入社1週目 | オリエンテーション + 座学研修（PC動画） |
| 入社2〜4週目 | OJT（担当者つき）+ 送迎同乗研修 |
| 入社1ヶ月 | 管理者との1on1面談（困りごとヒアリング） |
| 入社2ヶ月 | OJT卒業判定 + 担当業務の段階的拡大 |
| 入社3ヶ月 | 管理者との1on1面談（定着確認・今後の目標設定） |

### 継続的な定着施策

- 月1回の1on1（主任 or 管理者と）
- 半年に1回のキャリア面談（昇格・スキルアップの方向性）
- 「言いにくいこと」を書ける匿名アンケート（半年に1回）

## 8. KPI・効果測定

### 追うべき数値

| 指標 | 目標値 | 計測方法 |
|---|---|---|
| 表示回数 | 週500回以上 | Indeed管理画面 |
| クリック率（CTR） | 3%以上 | Indeed管理画面 |
| 応募数 | 月4件以上 | 全媒体合計 |
| 応募→面接率 | 50%以上 | 手動集計 |
| 面接→内定率 | 30%以上 | 手動集計 |
| 応募単価 | 1万円以下 | 費用 ÷ 応募数 |

### 数値の見方と打ち手

| 状態 | 原因の仮説 | 打ち手 |
|---|---|---|
| 表示回数が少ない | 予算不足 or 職種名が弱い | Indeed予算増額 / 職種名変更 |
| 表示はあるがクリックされない | 職種名・給与の訴求が弱い | タイトル・給与表記のABテスト |
| クリックはあるが応募されない | 原稿の内容が刺さっていない | キャッチコピー・仕事内容の見直し |
| 応募はあるが面接に来ない | 初回連絡が遅い or 日程調整が手間 | 即レス徹底 / 日程候補を複数提示 |
| 面接するが辞退される | 条件面 or 面接の印象 | 面接内容の見直し / 条件改善 |
