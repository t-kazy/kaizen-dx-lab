========================================
  ポモドーロタイマー 起動手順（Windows）
========================================

【1回だけ必要な準備】
─────────────────────
1. Pythonをインストールする
   → https://www.python.org/downloads/
   → 「Download Python 3.x.x」ボタンをクリック
   → インストール時に「Add Python to PATH」に必ずチェックを入れる！

【毎回の起動方法】
─────────────────────
2. 「pomodoro.pyw」ファイルをダブルクリックする
   → これだけで起動します！

【機能】
─────────────────────
・作業(25分) / 短い休憩(5分) / 長い休憩(15分) の3モード
・スタート / 一時停止 / リセット / スキップボタン
・セッション完了時にビープ音
・🍅 完了したセッション数を表示
・📌「最前面ON/OFF」で常に画面手前に表示
・⚙「設定」で時間のカスタマイズ

【キーボードショートカット】
─────────────────────
  スペースキー : 開始 / 一時停止
  R キー       : リセット
