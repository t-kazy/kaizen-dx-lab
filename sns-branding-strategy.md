## 0. この戦略の土台 — MVV（MISSION・VISION・VALUES）

SNS ブランディングの全コンテンツは、カイトGROUP の MVV を起点に設計しています。
投稿に迷ったときは、ここに立ち返ってください。

### MISSION

> **子どもの未来を"ソウゾウ"する**
>
> 想像する、その子の未来を。創造する、その子と一緒に。

### VISION

> **すべての子どもに「帰ってこられる場所」を。**
>
> 子ども・親・カイトが三位一体でつながり、"できる"が積み重なった先に、自分の力で歩き出せる未来をつくる。

### VALUES（行動指針）

| 指針 | 意味 | SNS で伝えるとき |
|------|------|------------------|
| **備（ソナエル）** | すべては備えから始まる | 準備の仕組み・支援計画の裏側 |
| **灯（トモス）** | 自分の中に火を灯し、誰かの光になる | 志の原点・子どもの小さな変化 |
| **繋（ツナグ）** | 一人で抱えず、繋がる | チームワーク・保護者との連携 |
| **起（オコス）** | 自ら動き、変化を起こす | ボトムアップの文化・挑戦する姿勢 |
| **拓（ヒラク）** | まだ誰も歩いていない道を拓く | DX推進・新しい支援のかたち |

### なぜ今 SNS ブランディングが必要なのか — 三方良し

| 届けたい相手 | 現在の課題 | SNSで実現したいこと |
|-------------|-----------|-------------------|
| **保護者（利用者）** | 安心して任せられるかが見えにくい | 療育の姿勢・子どもの成長が伝わり「ここなら安心」と思える |
| **既存職員** | 自社の価値を客観的に感じる機会が少ない | 自分たちの仕事が発信され、誇りとエンゲージメントが高まる |
| **求職者** | オープニング以降、応募が伸びない | カイトの文化・働き方の魅力が伝わり「ここで働きたい」と思える |

## 1. ブランドコンセプト

### ポジショニング

**「子どもの未来をソウゾウする — 福祉 × DX の実践者」**

放課後等デイサービスを0→1で2拠点立ち上げ、全員正社員・仕組み化・DX推進で「子どもが成長できる場所」と「職員がずっと働きたい職場」を両立させている30代経営者が、現場のリアルと想いを等身大で発信する。

### なぜこのポジションが強いか

| 要素 | 説明 |
|------|------|
| 希少性 | 「福祉 × DX × 全員正社員 × 0→1経営」を全部語れる人がほぼいない |
| 物語性 | 祖父から受け継いだ福祉への想い、大阪からのUターン、感情で動いた決断——人を惹きつけるストーリーがある |
| 時代性 | 福祉業界のDX化・人材確保は国策レベルの課題。発信すれば共感と需要が自然に集まる |
| 実績 | 2拠点の立ち上げ、年間休日120日・残業ほぼゼロの実現、属人性排除の仕組み化 |
| 哲学 | MVVに裏打ちされた一貫した世界観。「なぜやるか」がブレない |
| 人柄 | 等身大トーンにより「この人なら相談できそう」という信頼感 |

## 2. ターゲット設定

MVV策定の背景にある「三方良し」を起点に、3つのターゲットを設定します。
すべてのターゲットに対して、カイトGROUPの世界観・想いが一貫して届く発信を行います。

### ターゲット A: 求職者（最重要 — 採用の壁を突破する）

**「福祉業界で働く場所を探している20〜30代」**

- 保育士・児童指導員・作業療法士などの資格保持者
- 今の職場の人間関係・働き方に不満を抱えている
- 「ずっと働きたい」と思える職場を探している
- → **カイトの文化（全員正社員・年休120日・残業ゼロ・ボトムアップ）を知れば「ここで働きたい」に変わる**

響くメッセージ：
- 全員正社員、パート・バイトなし
- 年間休日120日、残業0〜1時間/月
- 現場への権限移譲、意見が言いやすい環境
- 仕組み化・IT化で「ムリ・ムダ・ムラ」を排除
- 一緒に組織をつくっていける楽しさ

### ターゲット B: 保護者（利用者獲得 — 安心と信頼の構築）

**「発達に特性のある子どもの放デイを探している保護者」**

- 小学生の子どもを持つ保護者が中心
- 「安心して任せられるか」「子どもが楽しく通えるか」「成長できるか」を重視
- 施設のSNSを見て雰囲気を確認してから見学を申し込む
- → **療育の姿勢・子どもの変化・職員の人柄が伝われば「ここなら安心」に変わる**

響くメッセージ：
- 支援計画の緻密さ（やるべきことを当然のようにやっている）
- 子どもの成長を丁寧に見守る姿勢
- 「カイトに通って子どもが成長した」「学校は行けないけどカイトには行ける」という声
- 子ども・親・カイトの三位一体で支える安心感

### ターゲット C: 同業経営者・開業希望者（ブランド認知 — 業界での存在感）

**「福祉業界で独立・開業を考えている30〜40代」「すでに経営しているが業務効率化に悩む経営者」**

- 放デイや児童発達支援の開業を検討中、または既に運営中
- 人材確保・業務効率化・組織づくりに課題を感じている
- → **カイトの経営ノウハウ・DX実践が参考になり、講演・コンサル案件に直結する**

響くメッセージ：
- 0→1で2拠点立ち上げのリアル
- 全員正社員経営の仕組み
- IT・DX化による業務効率化
- 離職を経験して学んだ組織づくり

## 3. プラットフォーム戦略

### Instagram と Threads の使い分け

| | Instagram | Threads |
|---|---|---|
| 役割 | **メインの資産構築** | **日常の接点づくり** |
| コンテンツ | カルーセル投稿・リール（ストック型） | テキスト投稿・考え・日常（フロー型） |
| 頻度 | 週3〜4回 | 毎日1〜2回 |
| 目的 | 専門性の証明、検索流入 | ファンとの関係構築、拡散 |
| 時間配分 | 1日20〜30分 | 1日10〜15分 |

### Instagram 運用方針

**投稿フォーマットの優先順位：**

1. **カルーセル投稿（最優先）** — 5〜10枚のスライド形式。情報量が多く保存されやすい。動画編集不要
2. **リール（短尺動画）** — 顔出しで30〜60秒。まずはカルーセルに慣れてから段階的に導入
3. **ストーリーズ** — 日常の一コマ、アンケート、質問箱。フォロワーとの距離を縮める

**なぜカルーセルを最優先にするか：**
- 動画編集未経験でもCanva等で作成可能
- 保存数が伸びやすく、アルゴリズムに有利
- 情報を構造的に伝えられるため専門性を訴求しやすい
- 「ノウハウ系」と相性抜群

### Threads 運用方針

**投稿タイプ：**
- 今日の気づき・学び（140文字程度）
- 福祉現場の「あるある」
- AI/DXを現場に導入したときのビフォーアフター
- 経営で感じたこと（等身大トーン）
- Instagramの投稿を深掘りするテキスト

投稿は以下の5カテゴリをローテーションします。
各柱は VALUES の行動指針と対応させ、発信全体に MVV の世界観が自然に染み出すよう設計しています。

**目的：** 保護者の安心・信頼 ＋ 求職者への「ここはちゃんとしている」訴求
**主なターゲット：** B（保護者）、A（求職者）

投稿例：
- 「放デイの支援計画、うちはこう作っています」
- 「子どもが来る前の"準備"が療育の質を決める」
- 「個別支援計画のフロー全公開 — 4ステップで属人化を防ぐ」

- 「仕組みとルールで"誰がやっても同じ品質"を実現する方法」

形式：Instagram カルーセル（5〜10枚）+ Threads で補足テキスト

> **VALUES との接続：** 「すべては備えから始まる」。全員正社員へのこだわり、事前準備の徹底、仕組み化——カイトの根幹を伝える柱。

- 「祖父から受け継いだもの、カイトという名前に込めた想い」

形式：Instagram（カルーセル or リール）+ Threads テキスト

> **VALUES との接続：** 「自分の中に火を灯し、誰かの光になる」。子どもの中にある小さな火種を見つけ育てる。代表の原体験と志。

**目的：** 求職者への直接訴求 ＋ 既存職員のエンゲージメント向上

> **VALUES との接続：** 「一人で抱えず、繋がる」「自ら動き、変化を起こす」。ボトムアップ組織、風通しの良さ、チームプレーの文化を伝える。

**目的：** 専門性の証明 ＋ 同業経営者への訴求 ＋ 業界での権威性

### 5-3. 制作コスト比較（初心者の目安）

| 種類 | 制作時間（初心者） | 慣れた後 | 必要スキル | 必要ツール |
|------|-------------------|---------|-----------|-----------|
| **静止画** | 15〜30分 | 5〜10分 | Canva基本操作 | Canva（無料） |
| **カルーセル** | 60〜90分 | 15〜30分 | Canva基本操作 | Canva（無料） |

| **ストーリーズ** | 5〜10分 | 2〜5分 | スマホで撮って文字入れ | Instagramアプリだけ |

### 5-4. 総合評価マトリクス

```
              効果（リーチ＋エンゲージメント）
               高
               │
     リール     │  ★ カルーセル
    (しゃべり)  │
               │
               │         ★ リール
               │          (テキスト動画)
               │
               │  ★ ストーリーズ
               │                    ★ 静止画
               │
              低├──────────────────────────→ 高
              低    制作の手軽さ（工数が少ない）
```

| 評価 | 種類 | 理由 |
|------|------|------|
| **◎ 最推奨** | **カルーセル** | 効果トップクラス × Canvaで作れる × テンプレ再利用で時短 × 動画スキル不要 |
| **○ 推奨** | **ストーリーズ** | 超手軽 × フォロワーとの関係維持 × 「日常の裏側」を見せるのに最適 |
| **○ 推奨** | **リール（しゃべり系）** | リーチ力が圧倒的 × 顔出しで信頼構築 × 編集は最低限でOK |
| △ 余裕が出たら | リール（テキスト動画） | 効果は高いが編集工数も高い |
| △ 余裕が出たら | ライブ配信 | フォロワーとの濃い接点になるが、ある程度のフォロワー数が必要 |
| ▲ 優先度低 | 静止画1枚 | 効果が低い割に手間は変わらない。カルーセルの方が上位互換 |

### 5-5. 初心者おすすめロードマップ

#### フェーズ1: まずはこれだけ（1〜2ヶ月目）

```
週の投稿メニュー（合計 3〜4投稿）：

  月  ─  カルーセル投稿 ★メイン
  水  ─  カルーセル投稿 ★メイン
  金  ─  ストーリーズ（日常の一コマ）
  土  ─  ストーリーズ（日常の一コマ）
```

**なぜカルーセル × ストーリーズから始めるのか：**

- **カルーセル** = エンゲージメント・保存数が全形式中トップ。Canvaのテンプレ使い回しで2回目以降は15〜30分で作れる。動画編集スキル不要
- **ストーリーズ** = 5分でできる。フォロワーとの距離を縮める。カルーセルの新投稿告知にも使える
- この2つだけで **「発見される力」と「ファンを作る力」の両方をカバー** できる

#### フェーズ2: リールを追加（3ヶ月目〜）

```
週の投稿メニュー（合計 4〜5投稿）：

  月  ─  カルーセル投稿
  火  ─  ストーリーズ
  水  ─  リール（しゃべり系） ★追加
  木  ─  ストーリーズ
  金  ─  カルーセル投稿
```

- カルーセルで使ったネタの「要約版」をリールにすれば、ネタの使い回しもできる

## 6. 投稿テンプレート

### そもそもカルーセルとは？

Instagramの「カルーセル投稿」とは、**1つの投稿に複数の画像（最大20枚）をまとめて、横スワイプで読める形式**のこと。

```
イメージ：
 ┌─────────┐   ┌─────────┐   ┌─────────┐
 │  1枚目  │ → │  2枚目  │ → │  3枚目  │ → ...
 │ （表紙） │   │（本文1）│   │（本文2）│
 └─────────┘   └─────────┘   └─────────┘
      ← 指で横にスワイプして読む →
```

**なぜカルーセルが強いのか：**
- 紙芝居のように「次も見たい」とスワイプさせるので滞在時間が伸びる
- 滞在時間が長い投稿はInstagramのアルゴリズムに評価される → より多くの人に表示される
- 「あとで見返そう」と保存されやすい → 保存数もアルゴリズムに有利
- 動画編集のスキル不要。Canvaで画像を並べるだけで作れる

### Instagram カルーセルの型

**黄金構成（10枚テンプレ）：**

| 枚数 | 内容 | 例 |
|------|------|------|
| 1枚目 | **表紙（キャッチコピー）** | 「放デイの書類業務をAIで半分にした方法」 |
| 2枚目 | 問題提起 | 「こんなお悩みありませんか？」 |
| 3枚目 | 共感 | 「実は私もそうでした」 |
| 4〜7枚目 | 解決策・ノウハウ（本編） | 具体的な手順やポイントを1枚1メッセージで |
| 8枚目 | まとめ | 要点を3つに整理 |
| 9枚目 | 自己紹介 | ブランドステートメント + 実績 |
| 10枚目 | CTA（行動喚起） | 「保存して見返してね」「フォローで最新情報をチェック」 |

### Canva でカルーセルを作る手順（初心者向け・ステップバイステップ）

#### STEP 1: デザインを新規作成する

1. [canva.com](https://www.canva.com) を開いてログイン
2. 右上の「デザインを作成」ボタンをクリック
3. 検索欄に **「Instagram 投稿」** と入力
4. **「Instagramの投稿（正方形）」（1080×1080px）** を選択
   - ※ カルーセル専用のサイズはないので、正方形の投稿サイズでOK

#### STEP 2: 表紙（1枚目）を作る

**これが一番大事。フィードで目に止まるかどうかはここで決まる。**

1. 左メニュー「素材」→ 四角形の図形を追加 → 画面全体に広げる
2. 図形の色を **ネイビー `#1B365D`** に変更（カラーピッカーでHEXコード入力）
3. 左メニュー「テキスト」→「見出しを追加」をクリック
4. タイトルを入力（例：`放デイの書類業務を AIで半分にした方法`）
5. 文字色を **ホワイト `#FFFFFF`** に変更
6. フォントは **「Noto Sans JP」の Bold** がおすすめ（読みやすい）
7. 文字サイズは **36〜48pt** 程度（スマホで読める大きさ）
8. オレンジ `#FF6B35` の細い横線を入れるとアクセントになる

```
┌──────────────────────┐
│    ■ ネイビー背景     │
│                      │
│  放デイの書類業務を   │  ← 白文字・太字
│  AIで半分にした方法   │
│  ━━━━━━━━━━━━━━━━  │  ← オレンジのライン
│                      │
│  梶井たいせい         │  ← 小さめの白文字
│  放デイ経営×DX        │
└──────────────────────┘
```

#### STEP 3: ページを追加して中面（2〜8枚目）を作る

1. 画面下部の **「＋ ページを追加」** をクリック（これで2枚目ができる）
2. 同じ操作を繰り返して必要な枚数分ページを追加
3. 中面の基本レイアウト：

```
┌──────────────────────┐
│ ■ ネイビーの帯        │  ← 上部に見出し帯
│  見出しテキスト       │     （高さは画面の15%くらい）
│━━━━━━━━━━━━━━━━━━━━│
│                      │
│  □ 白 or ライトグレー │  ← 本文エリア
│    の背景             │
│                      │
│  ・本文テキスト       │  ← ダークグレー文字
│  ・強調したい所は     │
│    オレンジ文字       │
│                      │
│         ▶ 3/10       │  ← 右下にページ番号
└──────────────────────┘
```

4. 見出し帯の作り方：四角形を追加 → ネイビーに → 上部に配置
5. 見出しテキスト：白文字 で帯の上に配置
6. 本文：ダークグレー `#333333` で記入。強調はオレンジ `#FF6B35`

#### STEP 4: CTA（最終ページ）を作る

```
┌──────────────────────┐
│    ■ オレンジ背景     │
│                      │
│   この投稿が          │
│   役に立ったら        │  ← 白文字
│   保存 🔖 してね！    │
│                      │
│   ＠kajii_taisei     │  ← アカウント名
│   フォローもお願い！  │
└──────────────────────┘
```

1. 背景をオレンジ `#FF6B35` に
2. 文字はホワイト `#FFFFFF`
3. 自分のアカウント名を入れておく（フォロー導線）

#### STEP 5: ダウンロードしてInstagramに投稿

1. 右上の **「共有」→「ダウンロード」** をクリック
2. ファイルの種類：**PNG（推奨）** を選択
3. 「すべてのページ」にチェック → ダウンロード
4. スマホに画像を送る（AirDrop、Googleフォト、LINE の「自分だけのグループ」等）
5. Instagramアプリを開く → 「＋」→「投稿」
6. 画像を **枚数の順番通りに複数選択**（1枚目→2枚目→...の順）
7. キャプション（説明文）を書く → ハッシュタグを付ける → 投稿！

#### 時短テクニック: テンプレの使い回し方

**一度作ったデザインは毎回ゼロから作らない。**

1. 完成したデザインを開く
2. 右上「…」→ **「コピーを作成」**
3. コピーしたデザインのテキストだけ差し替える → 新しい投稿の完成

これで **2回目以降はテキスト差し替えだけなので10〜15分で作れる** ようになります。

### Threads 投稿の型

```
【型A: 気づき共有】
今日、〇〇してみたら△△だった。
福祉現場って□□だと思われがちだけど、
実は××すると劇的に変わる。

【型B: ビフォーアフター】
Before: 〇〇に毎日2時間かかってた
After: AIで△△して30分に短縮
やったこと → □□（3行で説明）

【型C: 問いかけ】
福祉業界の〇〇、皆さんどうしてますか？
うちは△△でやってるんですが、
もっと良い方法あったら教えてほしい。

【型D: ストーリー】
2店舗目を出すと決めた日の話。
正直、めちゃくちゃ怖かった。
でも〇〇と思って踏み出した。
（続きはプロフィールのリンクから）
```

## まとめ

この戦略の核心は **カイトGROUPのMVVを起点に、三方良し（保護者・職員・求職者）すべてに響く発信をすること** です。

「福祉 × DX」という唯一無二のポジションに加え、祖父から受け継いだ想い、凧（カイト）の比喩、全員正社員へのこだわり、備・灯・繋・起・拓の行動指針——これらのストーリーがSNS発信の全体を貫く世界観になります。

SNS未経験でも大丈夫です。以下の3原則を守ってください：

1. **完璧を求めず、まず出す** — 最初の10投稿は練習。質より量でリズムを掴む
2. **テンプレを使い倒す** — 毎回ゼロから作らない。型に沿って効率よく
3. **継続こそ最大の差別化** — 週3投稿を6ヶ月続けるだけで上位5%に入れる

迷ったときはセクション0のMVVに立ち返ってください。「この投稿はMISSION（子どもの未来をソウゾウする）に繋がっているか？」——その問いが、ブレない発信の羅針盤になります。

フットワークの軽さが最大の武器です。この戦略書をもとに、まず今日の3ステップから始めましょう。
