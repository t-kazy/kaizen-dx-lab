## 1. ブランドコンセプト

### ポジショニング

**「福祉 × DX の実践者」**

放課後等デイサービスを0→1で2店舗立ち上げた30代経営者が、福祉現場のリアルとAI/DXによる変革を等身大で発信する。

### なぜこのポジションが強いか

| 要素 | 説明 |
|------|------|
| 希少性 | 「福祉 × IT/DX × 0→1経営」を全部語れる人がほぼいない |
| 時代性 | 福祉業界のDX化は国策レベルで推進されており需要が爆増中 |
| 実績 | 2店舗の立ち上げ実績があるため机上の空論にならない |
| 人柄 | 等身大トーンにより「この人なら相談できそう」という信頼感 |

### ブランドステートメント（自己紹介テンプレ）

```
放デイ2店舗を0から立ち上げた30代経営者。
福祉の現場をAIとDXで変える実践記を発信中。
経営のリアル｜福祉×テック｜放デイ開業ノウハウ
```

## 2. ターゲット設定

ヒアリング時点では「まだ明確でない」とのことだったので、ブランドの方向性から逆算して設定します。

### プライマリーターゲット（最重要）

**「福祉業界で独立・開業を考えている30〜40代」**

- 放デイや児童発達支援の開業を検討中
- 現在は別の福祉事業所に勤務 or 異業種からの参入を考えている
- 0→1の立ち上げ方が分からず情報を探している
- → **この層がコンサル案件に直結する**

### セカンダリーターゲット

**「すでに福祉事業所を経営していて業務効率化に悩んでいる経営者」**

- スタッフ管理、書類業務、保護者対応に追われている
- AI/DXに興味はあるが何から手をつけていいかわからない
- → **講演案件・DXコンサルに直結する**

### ターシャリーターゲット

**「福祉業界に興味のある同世代の若手ビジネスパーソン」**

- 社会課題に関心がある
- 起業に興味があり、異業種参入のモデルケースとして見る
- → **ブランド認知の拡大、フォロワー基盤の構築**

## 3. プラットフォーム戦略

### Instagram と Threads の使い分け

| | Instagram | Threads |
|---|---|---|
| 役割 | **メインの資産構築** | **日常の接点づくり** |
| コンテンツ | カルーセル投稿・リール（ストック型） | テキスト投稿・考え・日常（フロー型） |
| 頻度 | 週3〜4回 | 毎日1〜2回 |
| 目的 | 専門性の証明、検索流入 | ファンとの関係構築、拡散 |
| 時間配分 | 1日20〜30分 | 1日10〜15分 |

### Instagram 運用方針

**投稿フォーマットの優先順位：**

1. **カルーセル投稿（最優先）** — 5〜10枚のスライド形式。情報量が多く保存されやすい。動画編集不要
2. **リール（短尺動画）** — 顔出しで30〜60秒。まずはカルーセルに慣れてから段階的に導入
3. **ストーリーズ** — 日常の一コマ、アンケート、質問箱。フォロワーとの距離を縮める

**なぜカルーセルを最優先にするか：**
- 動画編集未経験でもCanva等で作成可能
- 保存数が伸びやすく、アルゴリズムに有利
- 情報を構造的に伝えられるため専門性を訴求しやすい
- 「ノウハウ系」と相性抜群

### Threads 運用方針

**投稿タイプ：**
- 今日の気づき・学び（140文字程度）
- 福祉現場の「あるある」
- AI/DXを現場に導入したときのビフォーアフター
- 経営で感じたこと（等身大トーン）
- Instagramの投稿を深掘りするテキスト

## 4. コンテンツ戦略

### 4つのコンテンツ柱

投稿は以下の4カテゴリをローテーションします。

#### 柱1: 福祉 × DX 実践記（メイン柱・40%）

**目的：** 専門性の証明、検索流入、保存数の獲得

投稿例：
- 「放デイの書類業務をAIで半分にした方法」
- 「福祉現場に ChatGPT を導入してみた結果」
- 「連絡帳をデジタル化したら保護者満足度が上がった話」
- 「スタッフのシフト管理をDXしたらこう変わった」

形式：Instagram カルーセル（5〜10枚）+ Threads で補足テキスト

#### 柱2: 放デイ開業・経営のリアル（30%）

**目的：** ターゲット層への直接訴求、コンサル案件への導線

投稿例：
- 「放デイ開業に必要な資金、リアルな数字を公開」
- 「2店舗目を出すタイミングの判断基準」
- 「行政への申請で詰まったポイントと解決策」
- 「スタッフ採用で失敗して学んだこと」
- 「開業1年目にやってよかった3つのこと」

形式：Instagram カルーセル + Threads で裏話

#### 柱3: 個人のストーリー（20%）

**目的：** 人柄を見せる、親近感の構築、ファン化

投稿例：
- 「なぜ福祉の世界に飛び込んだのか」
- 「0→1で店舗を立ち上げた時の正直な気持ち」
- 「経営者のリアルな1日ルーティン」
- 「挫折した時にどう乗り越えたか」
- 「30代で福祉経営者になるまでのキャリアパス」

形式：Instagram（カルーセル or リール）+ Threads テキスト

#### 柱4: 業界知識・ノウハウ共有（10%）

**目的：** 業界への貢献、権威性の確立

投稿例：
- 「放課後等デイサービスとは？5分でわかる解説」
- 「障害福祉サービスの報酬改定、ここだけ押さえればOK」
- 「保護者対応で大切にしている3つのマインド」

形式：Instagram カルーセル

## 5. 投稿テンプレート

### Instagram カルーセルの型（Canva で作成）

**黄金構成（10枚テンプレ）：**

| 枚数 | 内容 | 例 |
|------|------|------|
| 1枚目 | **表紙（キャッチコピー）** | 「放デイの書類業務をAIで半分にした方法」 |
| 2枚目 | 問題提起 | 「こんなお悩みありませんか？」 |
| 3枚目 | 共感 | 「実は私もそうでした」 |
| 4〜7枚目 | 解決策・ノウハウ（本編） | 具体的な手順やポイントを1枚1メッセージで |
| 8枚目 | まとめ | 要点を3つに整理 |
| 9枚目 | 自己紹介 | ブランドステートメント + 実績 |
| 10枚目 | CTA（行動喚起） | 「保存して見返してね」「フォローで最新情報をチェック」 |

**表紙デザインのポイント：**
- 背景色を統一（ブランドカラーを決める → 後述）
- 文字は大きく、一言で内容が伝わるように
- 顔写真をアイコン的に入れると信頼感UP

### Threads 投稿の型

```
【型A: 気づき共有】
今日、〇〇してみたら△△だった。
福祉現場って□□だと思われがちだけど、
実は××すると劇的に変わる。

【型B: ビフォーアフター】
Before: 〇〇に毎日2時間かかってた
After: AIで△△して30分に短縮
やったこと → □□（3行で説明）

【型C: 問いかけ】
福祉業界の〇〇、皆さんどうしてますか？
うちは△△でやってるんですが、
もっと良い方法あったら教えてほしい。

【型D: ストーリー】
2店舗目を出すと決めた日の話。
正直、めちゃくちゃ怖かった。
でも〇〇と思って踏み出した。
（続きはプロフィールのリンクから）
```

## 7. 週間投稿スケジュール

### 週間カレンダー（目安）

| 曜日 | Instagram | Threads | 所要時間 |
|------|-----------|---------|----------|
| **月** | カルーセル投稿（柱1: 福祉×DX） | 投稿の補足テキスト | 40分 |
| **火** | ストーリーズ（日常） | 気づき共有 | 15分 |
| **水** | カルーセル投稿（柱2: 経営のリアル） | 投稿の補足テキスト | 40分 |
| **木** | ストーリーズ（質問箱・アンケート） | 問いかけ投稿 | 15分 |
| **金** | カルーセル投稿（柱3 or 柱4） | ビフォーアフター | 40分 |
| **土** | リール（余裕があれば） | ストーリー投稿 | 20分 |
| **日** | 休み（翌週の準備） | 休み or 軽い投稿 | 15分 |

**ポイント：**
- 月水金に集中してカルーセルを投稿。これがブランドの「資産」になる
- 火木はストーリーズとThreadsで接点を維持（低負荷）
- 日曜に翌週のネタ出し・カルーセル下書きを30分程度

### 1日の作業フロー（30分〜1時間版）

```
【パターンA: カルーセル投稿日（40分）】
1. テーマ確認（5分）
   → ネタリストから今日のテーマを選ぶ
2. カルーセル作成（25分）
   → Canvaテンプレートに文章を流し込む
3. キャプション作成 & 投稿（5分）
   → テンプレートに沿って記入
4. Threads補足投稿（5分）
   → カルーセルの内容を2〜3行で要約

【パターンB: 軽い運用日（15分）】
1. ストーリーズ投稿（5分）
   → 今日の現場の一コマ、気づきを共有
2. Threads投稿（5分）
   → 型に沿ってテキスト投稿
3. コメント・DM返信（5分）
   → エンゲージメントを維持
```

## まとめ
